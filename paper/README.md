## Auxiliary CUI source

The auxiliary CUI predictions were generated by running the public UIT_Worker Concept Detection notebook for ImageCLEFmedical 2026. The Concept Detection training and inference code was not modified in this work. Its output was used only as an auxiliary source of candidate UMLS CUIs for the official test images. These predicted CUIs were then mapped to readable biomedical terms and inserted into the Qwen3-VL prompt as optional clinical hints for the Caption Prediction pipeline.
