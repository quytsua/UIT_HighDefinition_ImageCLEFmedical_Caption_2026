# Auxiliary Concept Detection source

The auxiliary CUI predictions used by this Caption Prediction system were generated by running the public UIT_Worker Concept Detection notebook:

```text
https://github.com/nhanbui246/ImageCLEFmedical-2026-UIT-Worker
```

The UIT_Worker Concept Detection system is associated with Thai Q. To, Dinh C. Pham, Nhan H. Bui, and Thien B. Nguyen-Tat. It was developed for the ImageCLEFmedical 2026 Concept Detection subtask.

In the UIT_HighDefinition Caption Prediction system:

- the Concept Detection training and inference code was not modified;
- the generated CUI predictions were used only as external auxiliary inputs;
- predicted CUIs were mapped to readable biomedical terms;
- these terms were inserted into the Qwen3-VL prompt as optional clinical hints;
- the caption generator was instructed to use the hints only when visually supported by the image.

No Concept Detection notebook, checkpoint, dataset image, or CUI submission file is redistributed in this repository.
