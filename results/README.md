# Results summaries

This directory contains lightweight result summaries only. It does not contain full generated captions, rank-level CSV files, final submission files, private logs, or the full official leaderboard CSV.

Files:

| File | Purpose |
|---|---|
| `official_result_summary.md` | Final run metadata and official score. |
| `official_leaderboard_top8.md` | Top-8 final leaderboard summary used for reporting. |
| `final_submission_validation.md` | Internal validation summary for the final submitted file. |
| `machine1_output_summary.md` | Machine 1 rank-level summary without per-image captions. |
| `auxiliary_cui_summary.md` | Summary of the auxiliary CUI prediction file. |
