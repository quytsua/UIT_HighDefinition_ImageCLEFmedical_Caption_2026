# UIT_HighDefinition at ImageCLEFmedical Caption 2026
Repository accompanying the working notes paper:
"UIT_HighDefinition at ImageCLEFmedical Caption 2026: Soft CUI-Guided Qwen3-VL Captioning"

## Overview
This repository contains materials related to our participation in the ImageCLEFmedical Caption 2026 challenge.
Our approach combines a DenseNet121-based concept detector with Qwen3-VL-8B-Instruct for medical image caption generation. Predicted medical concepts are incorporated as soft guidance during prompting, while a post-processing stage is applied to improve caption consistency and formatting.
